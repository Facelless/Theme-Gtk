# AllBlack XFCE4 Theme

**AllBlack** é um tema minimalista e elegante para o gerenciador de janelas **XFCE4 (xfwm4)**.  
O tema é focado em cores escuras com detalhes em branco, proporcionando um visual moderno e limpo para seu desktop.

## Recursos

- Fundo do título da janela: preto (#000000)
- Texto do título da janela ativo: branco (#FFFFFF)
- Texto do título da janela inativo: cinza claro (#AAAAAA)
- Bordas da janela: branco (#FFFFFF)
- Cores de hover dos botões: cinza escuro (#222222)
- Layout de botões: Minimizar, Maximizar, Fechar (configurável)
